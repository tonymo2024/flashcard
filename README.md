# Flashcard Kids

Flashcard Kids is an educational Android application designed to help children learn and memorize words through interactive flashcards. Each flashcard features a word on one side and an image on the other, making learning fun and engaging.

## Features

- **Two-Sided Flashcards**: Each card displays a word and an associated image.
- **Local Storage**: User-inputted words and images are stored locally for easy access.
- **Automatic Image Generation**: Generate images based on the words for enhanced learning.
- **User-Uploaded Images**: Users can upload their own images to customize their flashcards.
- **Camera Integration**: Capture images directly from the camera to use in flashcards.
- **Progress Tracking**: Monitor learning progress with detailed statistics.
- **Multiple Card Decks**: Create and manage multiple decks of flashcards.
- **Audio Pronunciation**: Listen to the pronunciation of words to aid in learning.

## Requirements

- Android 14 (API level 34) or higher
- Kotlin programming language

## Project Structure

The project is organized into several modules, including:

- **Data Layer**: Manages local storage and data access through Room database.
- **UI Layer**: Composed of various screens and components for user interaction.
- **ViewModel Layer**: Manages UI-related data in a lifecycle-conscious way.
- **Utilities**: Contains helper functions for image handling, audio playback, and camera integration.
